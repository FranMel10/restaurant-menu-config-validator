# Restaurant Config Validator

A validation engine designed to detect misconfigurations in restaurant ordering systems.

## 🚀 Features

- Detects products without tags
- Identifies orphan tags (tags without kitchen profiles)
- Finds kitchen stations without printers
- Detects conflicting routing rules

## 🧠 Why this matters

Misconfigured restaurant systems can cause:
- Orders not printing in the kitchen
- Delays in service
- Lost revenue

This tool helps prevent those issues before they happen.

## 🛠️ Tech Stack

- TypeScript
- Node.js / Next.js (planned)
- Supabase (planned)
- Vercel (deployment)
